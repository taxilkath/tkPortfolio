A purpose-built clinic management software for **HNB Skin Clinic**, tailored to digitize patient handling, appointment booking, clinical documentation, and billing operations. Designed as a cross-platform desktop application using **Electron.js**, **Bootstrap** the system integrates with a **Node.js** backend and a **MySQL** database for robust data handling and real-time updates via **Socket.IO**.
