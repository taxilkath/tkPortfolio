A purpose-built clinic management software for **HNB Skin Clinic**, tailored to digitize patient handling, appointment booking, clinical documentation, and billing operations. Designed as a cross-platform desktop application using **Electron.js**, **Bootstrap** the system integrates with a **Node.js** backend and a **MySQL** database for robust data handling and real-time updates via **Socket.IO**.


---

## ✨ Features

- 🧑‍⚕️ Full patient lifecycle management with detailed medical records
- 📆 Integrated appointment scheduling and SMS notification support
- 📝 Clinical documentation including digital prescriptions and consultation templates
- 💸 Automated billing with GST support and PDF invoice, sticker generation
- 📊 Dashboard with real-time KPIs (appointments, revenue, trends)
- 🗃️ Local storage and data persistence using Electron Store
- 📡 Real-time UI updates with Socket.IO
- 🧾 Daily earnings report and advanced invoice breakdown
- 🌐 Multilingual support (Gujarati & English)
- 💻 Built-in PDF viewer and print-ready financial documents
- 💾 Backup and restore data
- 🔒 GDPR compliance
